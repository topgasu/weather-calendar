1 기상청 API 활성화 (필수)

인증키만 받는다고 끝이 아닙니다! 아래 두 서비스를 반드시 [활용신청] 해야 데이터가 나옵니다.

📢 다음 링크를 각각 클릭하여 [활용신청] 버튼을 누르세요:
  • 1. 단기예보 조회 서비스 (클릭)
  • 2. 중기예보 조회 서비스 (클릭)
* 신청 후 상태가 '사용'으로 바뀌어야 합니다 (보통 즉시 승인)

확인이 끝나면 마이페이지 > 인증키 현황에서 인증키를 복사해두세요.

2 내 지역 설정값 찾기

동네 이름을 검색하여 코드에 넣을 숫자를 확인하세요.

✅ NX: 60 / NY: 127

✅ 기온 구역(REG_ID_TEMP): 11B10101

✅ 육상 구역(REG_ID_LAND): 11B00000

* 이 값들을 update_calendar.py 상단 설정 부분에 붙여넣으세요.

3 GitHub Secrets 등록

내 저장소의 Settings > Secrets and variables > Actions로 이동합니다.

New repository secret 버튼을 누르고 다음과 같이 입력하세요.

  • Name: KMA_API_KEY
  • Secret: (복사한 기상청 인증키 입력)
