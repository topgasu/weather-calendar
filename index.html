1 기상청 API 활성화 신청

기상청에서 데이터를 보내주도록 먼저 '활용신청'을 해야 합니다.

📢 아래 두 링크를 각각 열어 [활용신청] 버튼을 꼭 누르세요!
  • 단기예보 서비스 신청
  • 중기예보 서비스 신청

신청 후 마이페이지 > 인증키 현황에서 나의 인증키를 미리 복사해두세요.

2 내 동네 설정값 찾기

동네 이름을 입력하고 버튼을 누르면 코드에 넣을 숫자가 나옵니다.

👇 아래 값을 메모하거나 복사해두세요!

  • NX: 60
  • NY: 127
  • 기온구역(REG_ID_TEMP): 11B10101
  • 육상구역(REG_ID_LAND): 11B00000

3 GitHub 설정 및 코드 수정

1. 이 저장소를 Fork 하여 내 계정으로 가져옵니다.

2. 내 저장소의 update_calendar.py 파일을 열고 연필 아이콘(Edit)을 누릅니다.

3. 상단 설정 부분에 아까 2단계에서 찾은 좌표 값들을 붙여넣고 저장(Commit)합니다.

4. 저장소 Settings > Secrets > Actions에 KMA_API_KEY라는 이름으로 내 기상청 인증키를 등록합니다.

4 달력 앱에서 구독하기

이제 생성된 날씨 파일을 내 폰에 연결할 차례입니다.

  • 내 저장소 메인 화면에서 weather.ics 파일을 클릭합니다.
  • 화면 우측 상단의 [Raw] 버튼을 누릅니다.
  • 브라우저 주소창에 뜨는 주소를 그대로 복사합니다!
  • 아이폰/갤럭시 달력 앱의 '구독 캘린더' 메뉴에 그 주소를 붙여넣으세요.
✨ Tip: [Raw] 버튼을 누르면 나오는 복잡한 텍스트 화면의 주소가 가장 정확한 데이터 주소입니다.
